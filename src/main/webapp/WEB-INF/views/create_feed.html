Добавление ленты

Нажмите для получения информации

В поле для ввода URL ленты необходимо ввести адрес страницы, на которой агрегатор будет искать новости. Адрес обязательно начинается с протокола (http или https). Например: "http://e1.ru/news" (без кавычек). В поле "Строка правил" нужно ввести строку, определяющую список правил парсинга. Поскольку агрегатор поддерживает парсинг HTML и RSS, для каждой из разметок определён свой формат правил. В качестве разделителя используется символ "|".

Правила для HTML: HTML|<основной домен сайта (начинается с протокола)>|<префикс URL новости на странице с новостями>|<тег содержащий текст новости на странице с новостью>|<один из классов тега с текстом новости>

Правила для RSS: Достаточно указать URL ленты и строке правил ввести RSS.

Затем нажать кнопку "Добавить".

Примеры:
URL ленты: http://e1.ru/news, строка правил: HTML|http://e1.ru|/news/spool/news_id-|td|news
URL ленты: https://lenta.ru, правило парсинга: HTML|http://lenta.ru|/news/|div|js-topic__text
URL ленты: https://lenta.ru/rss, правило парсинга: RSS
